Import and export namespaces

For general information on external references, see Namespaces and external references.

Import namespaces

To import a package with namespaces, proceed as follows:

  1. Click New/Load in the toolbar.

  2. Click Import Package.

    â†’ An Explorer window opens.

  3. Navigate to the package ZIP file which contains a single namespace.

  4. Click Open to confirm your selection.

    â†’ After confirmation, the files inside the ZIP file will be validated.

  5. Click on files in the summary dialog to view the status of it.

  6. Click Copy to clipboard to copy the summary in a JSON-format file for further improvements.

  7. Click on one of the options to choose what happens with already existing namespaces.

    • Click Keep to keep the existing namespace.

    • Click Replace to replace the namespace in the Workspace.

  8. Click Import to move your files into the Workspace.

â†’ Now your namespaces are added to the recent model.

Export namespaces

To export a namespace, proceed as follows:

  1. Click Save in the toolbar.

    â†’ A drop-down menu appears.

  2. Click Export Namespace.

  3. Select the available namespace.

  4. Click Validate.

    â†’ After validation is done, a summary dialog similar to the one for namespace import will be displayed. If the validation has files with errors, the namespace cannot be exported.

  5. Click Export.

â†’ The chosen namespace will be exported.
